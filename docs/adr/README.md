# Architecture Decision Records (ADR)

このディレクトリには、tacokumo API システムの重要なアーキテクチャ決定を記録したADR（Architecture Decision Record）が含まれています。

## ADR Format

各ADRは以下の構造に従います：

```
# ADR-XXX: [タイトル]

## Status
[Proposed | Accepted | Deprecated | Superseded]

## Context
[決定の背景と問題の説明]

## Decision
[採用された解決策]

## Consequences
[この決定の結果生じる利益とリスク]
```
